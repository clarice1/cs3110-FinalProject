Module Polynomial

module Polynomial: sig .. end

Representation of a polynomial.

This module represents a polynomial of any degree. It also determines where on the complex plane will the polynomial diverge, and where it will converge.

type t

t is the type of polynomial

type bt

bt is the type of a polynomial with a cached upper bound after which divergence is guaranteed

val bound : t -> bt

bound p caches a bound after which divergence is guaranteed

val forget : bt -> t

forget p is p with the cached bound removed

val zero : t

zero is the zero polynomial

val eval : t -> Stdlib.Complex.t -> Stdlib.Complex.t

eval p z is the result of applying polynomial p to input z

val diff : t -> t

diff p is the derivative of p

val sum : t -> t -> t

sum p q is the polynomial p + q

val mul : t -> t -> t

mul p q is the polynomial pq

val from_roots : Stdlib.Complex.t list -> t
val bounded : t -> Stdlib.Complex.t -> Stdlib.Complex.t option

bounded p input is None if iterating polynomial p beginning at input is guaranteed to diverge and Some (eval p input) if p is not guaranteed to diverge

val bbounded : bt -> Stdlib.Complex.t -> Stdlib.Complex.t option

bbounded is like bounded, but more efficient since the bound is already cached

val from_list : Stdlib.Complex.t list -> t

from_list lst is the polynomial with coefficients in lst. The first element of lst is the coefficient of the highest degree of the polynomial, continuing in decreasing order of degree Requires: if lst is non-empty, the first element of lst is not 0

val to_list : t -> Stdlib.Complex.t list

from_list p is the list of coefficients of p

val get_bound : t -> float

get_bound p returns a radius outside of which iterating polynomial p is guaranteed to diverge
