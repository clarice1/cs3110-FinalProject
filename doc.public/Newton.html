Module Newton

module Newton: sig .. end

Fractal application of Newton's Method.

This module uses Newton's method as the function applied repeatedly to generate a unique fractal. Comes with full functionality of all our other fractals (i.e. line drawing, resizing, zoom, etc.).

val newton_fun : (Stdlib.Complex.t -> Stdlib.Complex.t) ->
       (Stdlib.Complex.t -> Stdlib.Complex.t) ->
       Stdlib.Complex.t list -> float -> Stdlib.Complex.t -> Stdlib.Complex.t option

newton_fun f f' roots tolerance z is None if z is within tolerance of a value in roots and is Some z-f(z)/f'(z) otherwise

val newton_fun_no_stop : (Stdlib.Complex.t -> Stdlib.Complex.t) ->
       (Stdlib.Complex.t -> Stdlib.Complex.t) ->
       Stdlib.Complex.t -> Stdlib.Complex.t

newton_fun_no_stop is like newton_fun but does not stop if a value is near a root.

val newton_color : Color.rgb list ->
       Stdlib.Complex.t list -> float -> int option * Stdlib.Complex.t -> Color.rgb

newton_color colors roots tolerance (n, z) is a color in colors satisfying |z - c < tolerance|, where c is the position at the corresponding location in roots, or black if no such root exists

val newton_with_defs : Stdlib.Complex.t list -> float -> int option * Stdlib.Complex.t -> Color.rgb

newton_with_defs is like newton_color but with a built-in default list of colors. Requires: at most 8 roots.

val full_newton : Stdlib.Complex.t ->
       Stdlib.Complex.t ->
       int -> int -> int -> Stdlib.Complex.t list -> float -> unit

full_newton ll ur iter width height roots tolerance manages all of Newton's method for an image of width width, height height, iter many iterations on a polynomial with roots in roots. There should be at most 8 roots.
