Module Matrix

module Matrix: sig .. end

Representation of a matrix.

This module represents a matrix and the data stored within it, including complex matrices.

type 'a t

'a t is the type of a matrix containing values of type 'a

type cx_t = Stdlib.Complex.t t

cx_t is the type representing complex numbers

val to_lst : 'a t -> 'a list list

to_lst m is a list of the rows of matrix m, each represented as a list

val init : int -> int -> (int -> int -> 'a) -> 'a t

init rows columns f is a matrix with rows many rows and columns many columns. Further, the value at the the ith row and jth column of this matrix is f i j

val rows : 'a t -> int

rows m is the number of rows in m

val columns : 'a t -> int

columns m is the number of columns in m

val cx_init : Stdlib.Complex.t -> Stdlib.Complex.t -> int -> int -> cx_t

complex_init ll ur rows columns is the complex matrix with rows many rows and columns many columns whose values are a grid of evenly spaced complex numbers with the value ll in its lower left corner and the value ur in its upper right corner Requires: ll.re < ur.re ll.im < ur.im rows, columns >= 2

val get : int -> int -> 'a t -> 'a

get i j m is the value in the ith row and jth column of matrix m Raises: Invalid_argument "(i, j)" if there is no value for that index

val mapi : (int -> int -> 'a -> 'b) -> 'a t -> 'b t

mapi f m is the matrix whose value at row i, column j is f i j (get i j m)

val iterate : ('a -> 'a) -> int -> 'a t -> 'a t

iterate f n m is the matrix with the same number of rows and columns as m but whose value at row i column j is f^n m(i, j), i.e. the n-fold composition of f with itself Requires: n is greater than 0

val iterate_with_stop : ('a -> 'a option) -> int -> 'a t -> (int option * 'a) t

iterate_with_stop f n m is the matrix with the same number of rows and columns as m but whose value at index row i, column j is (Some k, z) if 0 <= k <= n is such that -(z -> Option.bind z f)^k (Some m(i, j)) = Some z and -(z -> Option.bind z f)^(k+1) (Some m(i,j)) = None, or (None, f^n(z)) if no such k exists

val iterate_with_stop_2 : ('a -> 'a -> 'a option) -> int -> 'a t -> (int option * 'a) t

Like iterate_with_stop, but the function map also depends on its initial value. The input function should have as its first argument the initial value and its second the most recent value.
